# Crave Lab Thai Restaurant Website

Custom static microsite for Crave Lab Thai Restaurant in Eaglemont, Victoria.

## What is included

- One-page custom website built with plain HTML, CSS, and JavaScript
- Menu-first design direction tailored to the "Crave Lab" brand
- Research-backed address, phone, hours, and menu highlights
- Dynamic open/closed status based on `Australia/Melbourne`
- Ready for GitHub Pages deployment

## Public sources used

- Google Maps link supplied by the client
- Google Maps listing data
- Eaglemont Village business page
- Public menu image linked from Eaglemont Village

## Local preview

Open `index.html` in a browser, or run a simple static server from this folder.
