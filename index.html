Lab notes

A Thai spot that leans sharper, brighter, and more concise than the average local menu.

Neighbourhood footprint

The venue is listed inside Eaglemont Village and currently trades as Crave Lab Thai Restaurant on Google Maps while the village site labels it Crave Lab Thai Kitchen. Both references point to the same Silverdale Road address and phone number.

Food direction

The menu leans into Thai classics with a slightly focused lab-style edit: curries, stir fries, salads, soups, grilled chicken, barramundi fish, and polished single-dish wok favourites like Pad Thai, Pad See Ew, and Pineapple Fried Rice.

Public description

Eaglemont Village describes the food as genuine Thai cuisine that packs intense, vibrant flavours sure to make you crave seconds. That line feels consistent with the menu structure and the name itself, so the website carries that more modern, high-contrast energy.
